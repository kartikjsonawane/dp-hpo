Key Insight

If hyperparameter dimensions are approximately conditionally independent, the optimal configuration can be found by optimising each dimension greedily and sequentially — analogous to shortest-path problems in dynamic programming. Each partial configuration is memoised so shared subproblems are never re-evaluated.

Key Findings

✅ 90.7% evaluation reduction — DP-HPO uses 10 evaluations vs 108 for grid search across all datasets and seeds.

✅ Competitive accuracy — accuracy gap vs grid search is only 3.33pp on Heart Disease and 1.47pp on MNIST.

✅ Faster than random search — same 50% fewer evaluations (10 vs 20) with structured rather than unguided sampling.

✅ O(n·d) complexity — scales linearly with search space size, not exponentially like grid search.
